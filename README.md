# Banking System

A simple banking system built with Spring Boot and React.

## Features

### Authentication
- Register
- Login
- JWT Authentication
- Refresh Token

### Customer
- View Accounts
- Deposit
- Withdraw
- Transfer Money
- Transaction History

### Admin
- User Management
- Lock/Unlock User
- Grant/Revoke Admin Role
- View System Transactions

## Tech Stack

### Backend
- Java 21
- Spring Boot 3
- Spring Security
- JWT
- Spring Data JPA
- MySQL

### Frontend
- React
- Vite
- Axios

## Architecture

User -> React -> Spring Boot -> MySQL
